A lightweight FiveM script that updates Discord Rich Presence with the player’s name, server ID, and real-time street location.
<img width="900" height="25" alt="image" src="https://github.com/user-attachments/assets/88a4338d-7347-4e6f-a03a-522d170cb1a7" />



## Features

✔ Displays player name and server ID  
✔ Shows real-time street location  
✔ Detects intersections (cross streets)  
✔ Custom Discord Rich Presence images  
✔ Updates automatically every 10 seconds  
✔ Lightweight and optimized    

## Installation

1. Download the latest release
2. Place the folder in your `resources` directory
3. Add to your `server.cfg`

ensure dterp-rich-presents

## Requirements

None (Standalone)

## Configuration

Inside `client.lua`, edit the following: appId = 1234567890123456789
